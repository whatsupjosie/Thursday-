## TROUBLESHOOTING QUICK START

### Avatar doesn't appear, fallback shows instead
**Check:** DevTools Network tab → `/assets/avatar/manny.glb` status  
**Should be:** HTTP 200  
**If 404:** Asset path is wrong  
**See:** AVATAR_IMPLEMENTATION_GUIDE.md → TROUBLESHOOTING

### Animations don't work
**Check:** Console for animation clip names  
**Command:** `avatar.clips.map(c => c.name)`  
**Use:** Exact name from list (case-sensitive)  
**See:** AVATAR_QUICK_REFERENCE.md → Playing Animations

### Memory issues
**Check:** DevTools Memory tab → profiler  
**Should:** Return to baseline after unmount  
**Verify:** avatarMgr.dispose() called on cleanup  
**See:** HARDENING_REPORT.md → Memory Management

### Getting errors
**Check:** Browser Console → error message  
**Message tells:** Exactly what failed  
**Next:** Read error handling section in appropriate doc  
**See:** AVATAR_IMPLEMENTATION_GUIDE.md → TROUBLESHOOTING
